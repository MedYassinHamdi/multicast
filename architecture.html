ğŸ¯ How It Works

1. Sender broadcasts a message to the multicast group address (230.0.0.1:4446)
2. All receivers that have joined the group receive the same message simultaneously
3. No need to know individual receiver addresses - just send to the group!
4. Efficient - One transmission reaches multiple destinations
